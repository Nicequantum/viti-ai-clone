### MERLIN VOICE ASSISTANT
**Mercedes-Benz Warranty Stories · Shop Floor**

#### VOICE MODES — tap [hand icon] to switch

**TAP TO TOGGLE** (default)
- Tap mic → START · Tap mic → STOP
- Best for: longer notes, walking the bay

**PUSH-TO-TALK** (best in loud bays)
- Tap hand icon → HOLD mic while speaking → RELEASE to stop
- Best for: lifts, air tools, compressors

#### MIC IS LISTENING WHEN…
- Mic button glows / says "Stop"
- Panel shows: Bay noise · Confidence · live text
- **Bold** = locked in · *Italic* = still hearing
- Stopped after silence? → Tap **RETRY**

#### DAILY WORKFLOW
1. **OPEN RO** → tap your repair line
2. **DOCUMENT** → Technician Notes (voice or type)
3. **GENERATE** → "Generate warranty story"
4. **REVIEW** → edit → Copy for CDK or Download PDF

#### SAY THIS (examples)
- *Finding:* "Quick Test found fault code P0300."
- *Test:* "Cylinder 3 coil failed resistance test."
- *Repair:* "Replaced all four coils. Cleared codes."
- *Verify:* "Test drive confirmed repair."

#### ⌨️ MANUAL TYPING ALWAYS AVAILABLE
Every field accepts keyboard input. Voice is a shortcut — never block the repair.

*Chrome or Edge · Allow mic · Wi‑Fi needed for AI*
