## Printing instructions

### Paper and size

| Setting | Recommendation |
|---------|----------------|
| **Paper** | US Letter (8.5″ × 11″) white cardstock (65–110 lb) — holds up better before lamination |
| **Orientation** | Portrait |
| **Sides** | Double-sided — **flip on long edge** |
| **Color** | Color recommended (mic indicators use green/red) |
| **Copies** | 1 per bay tablet + 2 spares per shop |

### Trim size (after print, before laminate)

Trim to **8.0″ × 5.0″** (fits beside most rugged tablets) or **8.5″ × 5.5″** (half-sheet feel).

Leave **0.25″ bleed margin** if your print shop trims.

### Font sizes (when exporting to PDF or Word)

| Element | Font | Size | Weight |
|---------|------|------|--------|
| Main title (“Merlin Voice Assistant”) | Arial, Helvetica, or Calibri | **28–32 pt** | Bold |
| Section headers | Same | **14–16 pt** | Bold |
| Body / steps | Same | **11–12 pt** | Regular |
| Callout box (“Manual typing…”) | Same | **13–14 pt** | Bold |
| Fine print (URL, version) | Same | **8–9 pt** | Regular |
| QR label | Same | **9 pt** | Regular |

**Line spacing:** 1.15–1.25 for body text. **Avoid** going below 10 pt anywhere technicians must read at arm’s length.

### Layout tips

- **White space:** Keep at least 0.3″ padding inside laminate edge — lamination seals can obscure text at borders.
- **Contrast:** Black text on white; use **one** accent color only (Mercedes blue `#00A0E3` or brand black `#000`).
- **Scannability:** No paragraph longer than 2 lines. Use bullets and numbered steps only.
- **Icons:** If exporting to Canva/Word, use simple mic/hand/checkmark icons — not clipart.

### Lamination

| Setting | Recommendation |
|---------|----------------|
| **Thickness** | 5 mil (standard) or 7 mil (heavy bay use) |
| **Finish** | **Matte** — reduces glare under bay lights |
| **Seal** | 1/8″ sealed border on all sides |
| **Mounting** | Adhesive strip or tablet stand clip — do not block charging port |

### Double-sided print checklist

- [ ] Print **page 1** — [Front](./Bay-Reference-Card-Front.md)
- [ ] Reload paper — **flip on long edge**
- [ ] Print **page 2** — [Back](./Bay-Reference-Card-Back.md)
- [ ] Hold to light — verify alignment (front title upright when back support section is upright)
- [ ] Trim → laminate → mount at each bay

## Dealership customization (before print)

Replace every `[BRACKET]` placeholder on the back card:

| Placeholder | Example |
|-------------|---------|
| `[DEALERSHIP NAME]` | Mercedes-Benz of [City] |
| `[MERLIN URL]` | https://merlin.yourdealer.com |
| `[SM NAME / PHONE]` | Jane Smith · 555-0100 |
| `[IT NAME / PHONE]` | IT Help Desk · 555-0199 |
| `[QR CODE]` | Generate QR → `[MERLIN URL]/docs` or link to hosted Quick Start PDF |
